Wat is het verschil met de script tag boven in te laden in de head tag t.o.v onder inladen aan het einde van je HTML tag?
De script leest het HTML van boven naar beneden tot het punt waar de script wordt opgeroepen. Als de script wordt opgeroepen VOOR de HTML wat hoort te veranderen met javascript gaat de script niets doen.
Wat doet de attribuut “defer” in de script tag
Het leest eerst de volledige HTML voordat de script wordt uitgevoerd.
Wat is het verschil tussen HTML en JavaScript?
HTML het skelet van een webpagina. Met javascript kan je logica erin stoppen. Dit is het brein van de webpagina.
